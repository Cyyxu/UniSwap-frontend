# 前端项目启动说明

## 快速开始

### 1. 安装依赖

```bash
cd frontend
npm install
```

### 2. 启动开发服务器

```bash
npm run dev
```

前端将在 http://localhost:3000 启动

### 3. 确保后端服务已启动

后端服务地址：http://localhost:8109/api

## 功能说明

### 已实现的功能模块

1. **用户系统**
   - 用户注册/登录
   - 个人中心
   - 用户信息管理

2. **商品系统**
   - 商品列表浏览（支持搜索、排序、分页）
   - 商品详情查看
   - 商品购买
   - 商品管理（发布/编辑/删除）
   - 商品收藏

3. **订单系统**
   - 订单列表
   - 订单详情查看

4. **社区系统**
   - 帖子列表
   - 帖子详情
   - 发布帖子
   - 帖子点赞/收藏
   - 帖子搜索

5. **AI 功能**
   - AI 智能对话
   - 商品推荐咨询

6. **其他功能**
   - 公告通知
   - 私信功能
   - 收藏管理

## 技术特性

- ✅ 响应式设计，支持移动端和PC端
- ✅ 精美的 UI 设计，使用 Ant Design 组件库
- ✅ TypeScript 类型安全
- ✅ 统一的 API 请求处理
- ✅ JWT Token 自动管理
- ✅ 路由守卫（需要登录的页面自动跳转）
- ✅ 错误处理和提示

## 注意事项

1. 确保后端服务已启动（端口 8109）
2. 首次使用需要先注册账号
3. 部分功能需要登录后才能使用
4. JWT Token 会自动存储在 localStorage 中
